1️⃣ What is the difference between var, let, and const?
Difference brtween var, let, const:
var: Function scoped, it can be updated and redeclared.
let: Block scoped, it can be updated but not redeclared.
const: Block scoped, it cannot be updated or redeclared.


2️⃣ What is the spread operator (...)?
The spread operator is used to expand or unpack element from an array or object. it lets: copy an array or object, combine multiple arrays or objects, add new items easily, pass array values as separate arguments to a function.


3️⃣ What is the difference between map(), filter(), and forEach()?
Difference brtween map(), filter(), and forEach():
map(): Use when we need to transfrom data. It always returns a new array of the same lenght.
filter():Use when we need to extract a subset of data based on a specific condition.
forEach(): Use when we only want to perform an action or side effect for each element.


4️⃣ What is an arrow function?
An arrow function is shorter way to write a function in JavaScript.It uses a => instead of the word function.


5️⃣ What are template literals?
Template literals are a way to write strings in JavaScript that make it easier to include variables and create multiline text.
